### Installation
1. Clone repo:
   git clone https://github.com/Som-0108/fitness_tracker_app.git
2. Navigate to project:
   cd fitness_tracker_app/fitness_tracker_app
3. Install dependencies:
   npm install
4. Run dev server:
   npm run dev

### Features
- Onboarding with gender/age/weight input
- Dashboard screen
- Profile screen
- Workout plan screen
- Navigation bar
